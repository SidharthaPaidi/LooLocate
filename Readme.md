# LooLocate

A project to help users find nearby restrooms/toilets quickly and easily.

## Demo

[![LooLocate Demo](https://img.youtube.com/vi/yeEE2y9B7jc/0.jpg)](https://youtu.be/yeEE2y9B7jc)

Watch the demo video: [https://youtu.be/yeEE2y9B7jc](https://youtu.be/yeEE2y9B7jc)

## Features

- Find nearby restrooms based on your location
- View restroom details and ratings
- Get directions to the nearest facility

## Installation

```bash
git clone https://github.com/SidharthaPaidi/LooLocate.git
cd LooLocate
```
